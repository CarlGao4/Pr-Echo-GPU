## 1.0.0 - 2026-05-29

### Features
- Multi-frame temporal echo/blend effect (up to 300 frames each direction)
- GPU acceleration with CUDA and OpenCL backends
- CPU SIMD optimization with automatic SSE2/AVX2/AVX-512 dispatch
- Multiple blend modes: New On Top/Bottom, Add, Maximum, Minimum
- Window functions: Constant, Linear, Square Root, Square, Cosine, Smooth Step
- Configurable window falloff ratio and mapping
- Per-direction opacity control
- Frame shortage handling: Blank, Repeat, Solid Color
- Sync forward/backward settings option
- Dual host support: Adobe Premiere Pro and After Effects
- About dialog with version and source code info
